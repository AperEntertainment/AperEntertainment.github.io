λcommon

A library written in C++ with common features.

flash_on

Modern and flexible

λcommon is written in modern C++17 and it has a C wrapper.

check

Features rich
  • - OS Detection
  • - Terminal manipulation
  • - String manipulation
  • - Files manipulation
  • - URI manipulation
  • - Color manipulation
  • - And more!

settings

Cross-platform

λcommon works on multiple platforms: x86, x64 and ARM architectures with Windows, Linux and maybe Mac OSX.



Example

#include <lambdacommon/system/system.h>

using namespace std;
using namespace lambdacommon;

int main()
{
  terminal::setup();
  terminal::setTerminalTitle("λcommon - example");

  cout << "Starting lambdacommon example with " << terminal::CYAN << "lambdacommon" << terminal::RESET << " v" << lambdacommon::getVersion() << endl;
  cout << endl;
  cout << "Hello " << terminal::LIGHT_YELLOW << system::getUserName() << terminal::RESET << endl;
  // Sleep for 250ms
  system::sleep(250);
  cout << endl << "I like your CPU, it is " << terminal::LIGHT_GREEN << system::getProcessorName() << terminal::RESET << " right?" << endl;
  system::sleep(1000);
  // Clear all the content of the terminal.
  terminal::clear();
  cout << "Because I like my creators I want you to see their website!" << endl;
  system::openURI("https://aperlambda.github.io/");
  return 0;
}

Installation

Windows

Download the latest zip file of the compiled binaries and the source code for Windows on GitHub.

Linux

ArchLinux (and variants)

Install λcommon from the AUR.

From source

You will need to clone the repository of λcommon, then you'll build the sources with CMake and make (or MSBuild). λcommon will create a target named `install` and `uninstall` to simply install or uninstall the library.
